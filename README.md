# ANTweaker

## Installation

1. Install Python.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Build

To compile the `.py` file into an `.exe` using PyInstaller:

```bash
pyinstaller --onefile antweaker.py
```
